# Password
Verificador de Senha em JavaScript

Este projeto é um simples verificador de senha feito em JavaScript, executado no terminal com uso do módulo `prompt-sync`.

# Funcionalidade

O usuário deve digitar uma senha de 6 caracteres. O sistema verifica se a senha é igual a `"fgk37a"` e retorna:

- `ACESSO LIBERADO!` se for a senha correta.
- `ACESSO NEGADO!` se for uma senha incorreta.
